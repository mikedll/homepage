Here are some web applications I've made.

  • YClip3 - App to make and share YouTube clip compilations. WIP. Built with Node.js 8.12, React 16, and the YouTube IFrame Player API.
  • YScan8 - App to examine how much "fervor" a video on YouTube has, based on its likes, dislikes, and views. Runs on PHP 7.2, Laravel, and a little bit of Vue.js.
  • PVueInv - Demo app tracks purchase orders for a fictional business. Runs on Rails 5 and Vue.js.
  • React Experiments - Demo app that is supposed to be functionally the same as PVueInv above, but instead of using Vue.js for the front end, it uses React.js.
